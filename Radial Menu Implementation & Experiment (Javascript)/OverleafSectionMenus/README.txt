Overleaf Section Menus Installation Guide:

1) Download Google Chrome.
2) Go to chrome://extensions
3) Toggle "Developer mode" to "On" (Found top right).
4) Press "Load Unpacked" and select the directory containing this README file.
5) Turn on, and go to any Overleaf project.
6) If the menus fail to load in, try to resize the PDF viewer to a larger size, and try to refresh the page.
